#1. Remove or put # in front of any existing landscape code in the building (except the landscape cutting code).

#2. Copy the following lines into the intended building's code.
#Coordinate Set
#Set MainX = 0
#Set MainZ = 0

#Tranquility City Landscape
#<Include buildings/KW_Maps/Tranquility City.txt>

#3. After copying the above lines, remove all # in front of all lines except the ones saying "Coordinate Set" and "Tranquility City Landscape".
#4. Move the landscape cutting code to after the above lines.
#5. Before loading the building to see the appearance, the "MainX" and "MainZ" values MUST be changed to re-position the landscape.
